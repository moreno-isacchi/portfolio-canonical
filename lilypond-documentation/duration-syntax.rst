***************
Duration syntax
***************

Available duration numbers:

* ``1`` - whole note, *semibreve*
* ``2`` - half note, *minim*
* ``4`` - quarter note, *crotchet*, *semiminim*
* ``8`` - eighth note, *quaver*
* ``16`` - sixteenth note, *semiquaver*
* ``32`` - thirty-second note, *demisemiquaver*
* ``64`` - sixty-fourth note, *hemidemisemiquaver*
* ``128`` - hundred and twenty-eighth note, *semihemidemisemiquaver*
* ``256`` - two hundred and fifty-sixth note, *demisemihemidemisemiquaver*
* ``512`` - five hundred and twelfth note 
* ``1024`` - one thousand and twenty-fourth note

Additional duration commands:

* ``\breve`` - double whole note, *breve*
* ``\longa`` - quadruple whole note, *longa*
* ``\maxima`` - octuple whole note, *maxima*, *duplex longa*, *larga* (only available in ancient music notation)

Dotted notes:

Any duration number can be followed by dots. For example:

* ``4.`` - dotted quarter note
* ``4..`` - double dotted quarter note
* ``4...`` - triple dotted quarter note
