***********************
How to install LilyPond
***********************

This document guides you through the LilyPond installation process.

Ubuntu
^^^^^^

You can install LilyPond directly from the package manager.

From the command line, write:
::
    
  sudo apt install lilypond


To verify that LilyPond has been correctly installed, write:
::
  
  lilypond --version

Windows 
^^^^^^^

You can install LilyPond by downloading the Windows binary files from LilyPond `download page`_.

.. _download page: https://lilypond.org/doc/v2.24/Documentation/web/download

Copy the content of the ``.zip`` file into a directory of your choice.

To verify that LilyPond has been correctly installed, move to the installation directory and write:
::
  
  lilypond --version

macOS
^^^^^

You can install LilyPond using the package manager Homebrew.

From the command line, write:
::
    
  brew install lilypond


To verify that LilyPond has been correctly installed, write:
::
  
  lilypond --version
